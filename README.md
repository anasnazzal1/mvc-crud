# Knowledge Academy MVC CRUD Project

This project is a **Back-End MVC application** that demonstrates **CRUD (Create, Read, Update, Delete) operations** for managing data. It is developed as part of **Knowledge Academy** training to practice structured MVC architecture and database interaction.

## Project Overview
This project follows the **MVC (Model-View-Controller) pattern** to separate concerns and maintain a clean, organized codebase. The main focus is implementing CRUD operations on a database with proper routing, controllers, and views.

## Features
- Full **CRUD functionality** for core entities
- MVC architecture with separate Models, Views, and Controllers
- Data validation and error handling
- Simple user interface (optional)
- Modular and maintainable code
